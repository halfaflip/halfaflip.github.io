2.1 Why is the formula for determining the number of rows in a truth table related only to the number of inputs, and why is the formula 2n?

The formula for determining the number of rows in a truth table is related to the number of inputs because the number of inputs correlates to the number of outputs that they generate, which in turn create rows containing said data. The formula is 2n because it has to do with inputs and outputs. For example, 2 inputs (or an n value of 2) in the formula would be 2^2. There are two inputs and two outputs
